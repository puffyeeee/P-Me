# P&Me（Puffyの紹介サイト）

Puffyという名前のペットの紹介サイト「P&Me」を、**ワンページスクロール**＋**大きな写真**＋**フェードインアニメーション**で構成した静的サイトです。

## 目的 / ゴール
- Puffyの世界観（やさしい/ぬくもり/写真メイン）を見せる
- Sound（Apple Music等）、Things（ショップ）、SNSへ誘導できる導線を用意
- 後から画像・文言・リンクを簡単に差し替えできる構造

---

## 現在できている機能（実装済み）
- ワンページスクロール構成（Hero → About → Sound → Things → Moments → From Puffy → Contact → Footer）
- デザイン：**tea-drop.com 風の高級感（B案：上品で甘い）**
  - 背景は「ほぼ白」＋**パステルのマーブル（細かめ）**（スクロールで色が順番に出ず、常に混ざって見える固定レイヤー）
  - UIの装飾を抑え、写真とタイポが主役
  - 余白広め・文字小さめ寄り（ハイブランド感）
  - Momentsは整列グリッドで統一感を優先
- 写真を大きく見せるレイアウト（Heroはフルスクリーン）
- Sound：カードUIではなく、**ミニマルなリンクリスト（線＋字間＋ホバー統一）**に変更
- Things：ボタンではなく、**テキストリンク（Shop/Lookbook）**に変更。さらに Digital（壁紙配布）を追加
- Hero：CTAを最小化し、広告的な余白とタイポで見せる構成に調整
- Hero入場時に「おかえり。パーちゃん待ってたよ」を控えめにふわっと表示（prefers-reduced-motion対応）
- About：歓迎文言はHero側に寄せ、Aboutはキャラクター紹介に集中（字体（lead/body/meta）を高級寄りに調整、Todayボックスを撤去）
- スクロールでの**繊細なフェードイン表示**（IntersectionObserver + 段差ディレイ）
- 写真も文字と同様にふわっと出現（全体ブラーではなく、白い霞ベールがグラデーションで抜ける“フェードイン的”演出）
- スクロール時の“切り替わり感”を上品にする薄いヴェール（Scroll veil）
- セクション切替の一瞬の“白い発光”→落ち着くトランジション（luxury flash）
- スクロール停止時に要素が“最後に整う”マイクロモーション（scroll settle）
- シネマティック演出（A案）：Hero写真の超スローKen Burns／背景マーブルの超スロー漂い／見出しの細線アニメ／セクション切替ヴェール（ScrollTrigger連動）／パララックス／画像マスク（clip-path：有機的な斜めスワイプ）
- テキスト演出：リード/本文を**実際に行分割**して段差表示（SplitText風：行ごとに上へ抜ける＋ブラー解除＋字間が締まる）
- 質感演出：グレイン（ノイズ）レイヤー
- 慣性スムーススクロール：Lenis（ScrollTriggerと同期）
- 画像ホバー：ズーム中心から“光の移動（specular highlight）”中心へ
- 写真トーン統一のためのごく薄いフィルム（オーバーレイ）
- 画像ホバー（拡大・彩度調整・オーバーレイ表示）
- スムーズスクロール（CSS + JS補助）
- Moments：ギャラリー画像クリックで高級モーダル表示（左右/ESC/スワイプ対応、ベール＋暗転）。キャプションは“写真外（下の余白）”に配置して広告的に見せる。モーダル内の短文はPuffy世界観の仮ストーリー（後で差し替え可）
- Contact（おたより窓口）は現在「準備中」。世界観を崩さない形で、将来“非公開で受け取る”運用にする想定
- フッターに全セクションリンク + SNSアイコン（追加しやすい）
- レスポンシブ対応（スマホでレイアウト最適化）

## 次のおすすめ開発ステップ
1. **画像の差し替え**（最優先）
   - `images/` に画像を配置し、`index.html` の `src` を差し替え
2. **リンク設定**
   - `js/main.js` の `CONFIG.links` を実リンクに変更
3. Contactを外部フォームへ差し替え
   - 送信先URLを用意し、フォームactionの変更または埋め込みに変更

## デザイン方針（上質化）
- 背景（マーブル/グレイン）は維持したまま、説明感の強いUI（タグ/注釈/ボタン感）を削ぎ落として“作品感”へ寄せています。
- SNSは丸アイコン（円形ボタン）で表示。その他のUIは説明感を抑え、必要な箇所だけを上品に整理。
- Wallpapersはピル（タグ）ではなく、クレジットライン（小さな1行）で情報を静かに提示。

## 公開（検索OK）について
- 現在は通常公開モードです（検索エンジンへのインデックスを許可）。
- `index.html` の `noindex` は削除済み、`robots.txt` はクロール許可（Allow）です。

## 公開URL / API
- 公開（デプロイ）は **Publish タブ**から行ってください。
- 外部APIは使用していません。

---

## データモデル / 保存先
- データベースは使用していません（静的HTML/CSS/JSのみ）。
